# 🏨 Hotel Booking Analysis | Exploratory Data Analysis (EDA)

## 🚀 Project Overview

The Hotel Booking Analysis project focuses on exploring hotel reservation data to uncover booking patterns, cancellation trends, customer behavior, and revenue-related insights. Using Python and data visualization techniques, this project transforms raw booking data into actionable business intelligence that can help hotels improve occupancy rates and customer satisfaction.

---

## 🎯 Business Objective

The primary objective of this project is to analyze hotel booking data and identify key factors affecting booking cancellations, customer preferences, and hotel performance.

The analysis helps stakeholders:

- Understand customer booking behavior
- Identify cancellation trends
- Analyze seasonal demand patterns
- Improve revenue management strategies
- Support data-driven decision-making

## 📈 Key Insights

- City Hotels received more bookings than Resort Hotels.
- Higher lead times were associated with increased cancellation rates.
- Certain market segments generated the majority of bookings.
- Seasonal patterns significantly impacted booking demand.
- Average Daily Rate (ADR) varied across hotel types and reservation statuses.
